- **分层架构与循环依赖解耦**：采用 `handler` -> `service` (根包) -> `service/*` (子包) 的分层结构。通过 `service/hooks.go` 定义全局函数变量，并在各子包的 `init()` 中注册实现（如 `vm/migration/register.go`），有效解决了 Go 语言中的循环导入问题。
- **启动引导与状态恢复**：`main.go` 负责初始化配置、数据库及 Libvirt RPC 连接，并在启动阶段同步虚拟机缓存、恢复防火墙规则、桥接网络及端口转发等运行态配置，确保系统重启后状态一致。
- **统一异步任务调度**：内置 `taskqueue` 模块，在启动时集中注册包括克隆、迁移、快照、模板制作等在内的数十种长耗时任务处理器，实现业务逻辑与 HTTP 请求的解耦。
- **路由与中间件治理**：`router/router.go` 统一定义 API 端点，集成 JWT 鉴权、速率限制、CORS 及管理员权限校验等全局中间件，并支持前端 SPA 静态资源回退。
