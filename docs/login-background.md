# 登录页背景说明

## 本次调整

- 登录页背景图改为 `web/src/assets/login-bg-cloud-tech.png`
- 登录页样式文件位于 `web/src/views/login/index.vue`
- 保留了旧资源 `web/src/assets/login-bg.png`，未直接覆盖，方便后续对比或回退

## 视觉方向

- 主题：云服务器、云计算、虚拟化基础设施
- 风格：克制、稳重、偏企业级科技感
- 色调：深蓝、青蓝高光，避免花哨和高饱和杂色

## 接入方式

登录页容器通过以下方式叠加背景：

- 第一层：深色渐变遮罩，保证登录卡片可读性
- 第二层：科技风背景图
- 第三层：顶部径向光效和纵向暗化遮罩，减少画面发灰

## 后续替换建议

若后续需要替换登录页背景，建议：

1. 新图片优先保持横向大图比例，避免移动端裁切过于激进
2. 中央区域尽量保留一定留白，方便承载登录卡片
3. 不要直接覆盖旧图，建议新增文件后再修改 `index.vue` 中的背景引用
4. 若新图整体偏亮，需要同步调整 `login-container::before` 和 `login-box` 的透明度
