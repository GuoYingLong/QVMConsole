# 全局 ISO 存放位置

## 功能说明

系统设置的“存储与网络”标签页新增了 **ISO 存放位置** 配置项：

- 默认值：`/var/lib/libvirt/images/ISO`
- 对应环境变量：`KVM_ISO_DIR`
- 保存后立即生效，并会持久化到数据库

## 生效范围

该目录会同时影响以下功能：

- 管理员创建虚拟机时的全局 ISO 下拉列表
- 系统设置中的“救援系统 ISO”下拉列表
- `GET /api/storage-pool/all-isos`
- `GET /api/vm/iso-list`

系统只会扫描这个目录下一级的 `.iso` 文件，不再依赖历史的 libvirt `iso` storage pool 路径推断。

## 使用建议

- 如果宿主机原来的 ISO 放在旧目录，请把文件移动到新的 ISO 存放位置，或直接在系统设置中改成现有目录。
- 目录需要保证 `libvirt-qemu` 或 `qemu` 用户具备读取权限，否则虚拟机挂载 ISO 时可能失败。
- 新安装环境会自动创建默认目录并赋予与模板目录相同的基础权限。
