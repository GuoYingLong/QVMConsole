# 任务中心（异步任务管理）

## 功能概述

任务中心用于管理和监控所有耗时的异步操作。所有长时间运行的操作（如克隆、删除、创建虚拟机等）都会提交到任务队列，通过任务中心可以实时查看执行进度。

> **注意**：任务记录保存在内存中，不持久化到数据库。服务重启后任务记录会清空，超过 24 小时的已结束任务会自动清理。
>
> **权限范围**：管理员可查看和管理全部任务；普通用户只能查看、取消、清理自己创建的任务，实时推送也只会收到自己的任务事件。

## 支持的任务类型

| 类型标识 | 中文名称 | 触发场景 |
|---------|---------|---------|
| `clone` | 链式克隆 | 基于模板克隆虚拟机 |
| `linked_clone` | 原生链式克隆 | 管理员基于模板直接创建 backing qcow2 并启动 |
| `batch` | 批量克隆 | 批量克隆虚拟机 |
| `reinstall` | 重装系统 | 重装虚拟机系统 |
| `prepare` | 制作模板 | 将虚拟机制作为模板 |
| `create` | 普通创建 | 从 ISO 创建虚拟机 |
| `delete` | 删除虚拟机 | 删除虚拟机及其磁盘 |
| `vm_migrate` | 迁移虚拟机 | 管理员跨节点迁移 VM，目标面板接管，源节点保留副本 |
| `vm_disk_migrate` | 迁移硬盘 | 管理员将 VM 的指定硬盘迁移到本机目标存储 |
| `storage_format` | 格式化存储 | 格式化并挂载宿主机硬盘为存储池 |
| `storage_create_partition` | 创建分区 | 在宿主机硬盘上创建新分区 |
| `storage_delete_partitions` | 删除分区 | 卸载并删除宿主机硬盘上所有分区，清除分区表 |
| `runtime_quota_shutdown` | 运行时长配额关机 | 用户总运行时长配额耗尽后自动关闭其全部 VM |
| `lightweight_runtime_quota_shutdown` | 轻量云时长关机 | 单台轻量云 VM 运行时长配额耗尽后自动关闭该 VM |
| `enter_maintenance_mode` | 启用维护模式 | 关闭运行中的 VM/容器并停用维护服务 |
| `exit_maintenance_mode` | 关闭维护模式 | 恢复维护服务开机自启并尝试启动 |
| `vm_schedule_action` | 虚拟机定时任务 | 定时开机、定时关机、定时删除虚拟机 |

## 任务状态

| 状态 | 说明 |
|------|------|
| `pending` | 等待中，任务已提交等待执行 |
| `running` | 执行中，任务正在执行 |
| `success` | 成功，任务执行完成 |
| `failed` | 失败，任务执行出错 |
| `canceled` | 已取消，任务被用户取消 |

## 页面功能

### 异步任务面板（底部任务栏）

全局页面底部固定显示「异步任务」面板（类似 ESXi 的 Recent Tasks 栏），无需离开当前页面即可查看任务进度。

**面板功能**：
- **展开/收起**：点击标题栏的箭头按钮切换面板展开或收起状态
- **拖拽调整高度**：拖拽面板顶部的拖拽手柄可自由调整面板高度（最小 40px，最大为视口高度的 60%）
- **SSE 实时更新**：面板内任务进度通过 SSE 实时推送自动刷新
- **高度记忆**：面板高度和展开/收起状态自动保存，刷新页面后恢复
- **导航栏入口**：点击导航栏右侧的任务图标可切换面板展开/收起，有运行中任务时显示数量徽标
- **完整任务中心**：点击「完整任务中心」按钮可跳转到完整的任务列表页面（支持筛选、分页、详情等全部功能）

### 任务列表（完整页面）
- **实时进度显示**：通过 SSE（Server-Sent Events）实时推送任务进度，页面自动刷新进度条和状态消息
- **筛选功能**：支持按任务状态和任务类型筛选
- **分页**：支持分页浏览历史任务

### 操作
- **查看详情**：点击「详情」按钮打开抽屉，查看任务参数（JSON）和执行结果
- **取消任务**：可取消自己有权限访问且处于「等待中 / 执行中」的任务
- **清理任务**：管理员可清理全部已完成/失败/已取消任务；普通用户仅清理自己的已结束任务

### 重装系统任务说明

- 任务类型为 `reinstall`
- 提交后会先自动删除该 VM 的全部快照，再执行强制关机、替换系统盘和模板初始化
- Linux 模板会在开机后继续执行 SSH 初始化，因此重装任务耗时通常会比普通关机/开机更长
- Windows/FnOS 模板会在离线阶段完成大部分初始化；Windows 若启用了首次冷重启策略，任务中会包含一次额外的宿主机重新开机
- 任一步骤失败时，任务会尽最大努力回滚到原系统盘，并让 VM 保持关机

### 实时连接状态
- 页面右上角显示 SSE 连接状态指示器
- 连接正常时显示绿色「实时连接」标签
- 断线后自动重连（5秒间隔）

## API 接口

### 获取任务列表
```
GET /api/task/list?page=1&page_size=20&status=running&type=clone
```

参数：
- `page` - 页码（默认 1）
- `page_size` - 每页数量（默认 20，最大 100）
- `status` - 状态筛选（可选：pending/running/success/failed/canceled）
- `type` - 类型筛选（可选：clone/linked_clone/batch/reinstall/prepare/create/delete/vm_migrate/vm_disk_migrate）

返回范围：
- 管理员返回全部任务
- 普通用户仅返回当前用户创建的任务

### 获取任务详情
```
GET /api/task/:id
```
管理员可查看任意任务，普通用户仅可查看自己创建的任务。

### SSE 实时推送
```
GET /api/task/sse?token=<JWT Token>
```
通过 EventSource 订阅，接收 `task_progress` 事件。

推送范围：
- 管理员接收全部任务事件
- 普通用户仅接收自己创建的任务事件

事件数据格式：
```json
{
  "task_id": 1,
  "type": "clone",
  "status": "running",
  "progress": 50,
  "message": "创建链式克隆磁盘..."
}
```

### 取消任务
```
POST /api/task/:id/cancel
```
管理员可取消任意自己有权访问的等待中/执行中任务，普通用户仅可取消自己创建的等待中/执行中任务。

### 清理已完成任务
```
DELETE /api/task/clear
```
管理员清理全部 success/failed/canceled 状态的任务记录，普通用户仅清理自己的已结束任务记录。
