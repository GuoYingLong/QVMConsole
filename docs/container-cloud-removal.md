# 容器云移除说明

## 变更摘要

当前版本已移除容器云相关功能，面板现在仅保留虚拟机、存储、网络、防火墙、账户安全等能力。

已移除内容包括：

- 容器管理
- Docker Compose 编排
- 镜像拉取与共享
- Docker 卷与网络管理
- 容器带宽控制
- 容器存储页与相关接口
- 容器配额、镜像配额与容器流量统计

## 影响范围

- 前端侧边栏不再显示“容器云”入口。
- “我的存储”中不再显示“容器存储”标签。
- 用户配额中不再包含容器数量、镜像存储、共享镜像相关字段。
- 后端不再暴露容器云相关 API 路由。
- 维护模式仅收敛虚拟机与配置中的宿主机服务。

## 升级说明

- 已存在的容器云数据库表和历史数据不会再被面板使用。
- 如果宿主机上仍保留 Docker 或历史容器资源，需由管理员按实际运维策略自行处理。
- 如需继续使用面板，建议同步清理旧版容器云文档，统一以当前文档为准。
