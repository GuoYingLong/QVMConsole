# 虚拟网络入口移除说明

## 变更概述

系统已移除左侧「虚拟网络」菜单以及 `/api/virt-network/*` 相关接口，避免继续暴露与当前 OVS / VPC 网络方案重复的旧入口。

## 当前使用方式

- 管理员统一在左侧「网络」页面管理 OVS、VPC、安全组、端口转发与相关诊断。
- 普通用户统一在左侧「VPC 网络」页面管理自己的交换机与安全组。
- 虚拟机创建、模板克隆、磁盘导入时不再单独选择虚拟网络，系统会使用当前默认网络，并继续支持选择 VPC 交换机和安全组。

## 兼容说明

- 原有 `KVM_DEFAULT_NETWORK` 与系统设置中的默认网络字段仍保留，用于底层创建、克隆、导入流程的默认网络名称。
- 如需调整虚拟机的网络归属，请优先使用 VPC 绑定、网络管理页或系统默认网络设置，不再使用独立虚拟网络管理页。
