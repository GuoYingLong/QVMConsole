# 虚拟机列表模板列溢出处理

## 概述

虚拟机列表中的“模板”列在模板名称较长时，原先会因为单元格换行导致整行高度被撑高，影响列表密度和可读性。

本次调整后：

- 列表行高保持稳定
- 超长模板名称自动省略显示
- 鼠标悬停时显示完整模板名称

## 实现方式

前端统一使用 Element Plus 表格列的 `show-overflow-tooltip` 能力处理模板列溢出，不额外写死截断长度，也不依赖自定义计算逻辑。

## 涉及页面

- `web/src/views/vm/list.vue`
- `web/src/views/user/index.vue`
- `web/src/views/invite/index.vue`

## 说明

这次一并同步了其他同样展示模板名称的表格，避免不同页面对超长模板名称的处理方式不一致。
